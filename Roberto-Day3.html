Interpolation Methods

Our datasets are usually points in space. Interpolation “guesses” at what happens between them. It finds a function which goes through all the points. It generates a continuity between discrete observations. * Constant Interpolation: draws lines between points. Implemented through approxfun

  • Extrapolation: Extrapolation can have even more problems than interpolation. It’s really guessing based on an assumption (your interpolation function’s behaviour outside the data range).
curve(g, -1, 11, n=1000, col="blue")
curve(f, -5, 15, n=1000, col = "red", add=TRUE)

  • Linear Models: You can use the lm.fit function to calculate the linear model. You have to transform x and y into a matrix, with length(y) lines, x on the first column and 1 on the second.
myLm = lm.fit(cbind(a=x, b=1), y)
coef = myLm$coefficients
plot(x, y, xlim=c(-5, 15), ylim=c(-90, 150))
abline(coef[2], coef[1], col="red")

  • Other ideas: Let’s try second-degree. lm.fit does that as well.
myLm = lm.fit(cbind(a=x^2, b=x, c=1), y)
coef = myLm$coefficients
plot(x, y, xlim=c(-5, 15), ylim=c(-90, 150))
curve(coef[1]*x^2+ coef[2]*x+coef[3], -5, 15, n=1000, col="red", add=TRUE)

Root Finding

This is about numerically finding an x value where f(x) is zero (or very close to it). In general, it is about find any value of x where f(x) is closest to some y value.

  • Bisection: Depends on values at the edges of the interval having different signs. uniroot function does this. Write a function bisection() with the following parameters:
    • a real function f()
    • a numeric value a
    • a numeric value b > a
    • a positive real eps (defaults to 10-16)
    • a positive integer maxiter (defaults to 100)
bisection <- function(f, a, b, eps=1e-16, maxiter=100) {
    stopifnot(class(f) == "function")
    stopifnot(b>a)
    stopifnot(is.numeric(f(a)))
    stopifnot(eps>0)
    minInterv = a
    maxInterv = b
    meanInterv = (a+b)/2
    i = 1
    while (i <= maxiter && (abs(f(meanInterv)) >= eps)) {
        if (sign(f(meanInterv)*f(minInterv)) >= 0)
            minInterv = meanInterv
        else
            maxInterv = meanInterv
        meanInterv = (minInterv+maxInterv)/2
        i = i + 1
    }
    if (i > maxiter)
        warning("Max iterations reached before converging:")
    return(list(root=meanInterv, f.root=f(meanInterv), iter=i, estim.prec=(maxInterv-minInterv)/2))
}
